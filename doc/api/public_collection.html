UT.PublicCollection

An instance of UT.PublicCollection is:

  • writable by the world,
  • defined in expression.json file
  • contains 0 or 1 item per post viewer, no more,
  • only the current post viewer user's item can be modified.

The purpose of a public collection is to let users interact with the post (eg. by voting for one item or another) in some way. The public collection has the ability to support thousands of items and to be world writable. That means you cannot retrieve all items but only recent ones. You can however retrieve computed values like count, sum and average of a given fields. The items stored in public collections are objects instances whose fields have been declared to contain a certain type of values.

The collections are retrieved from the post instance using its collection(name) method.

Sample

First, you need to define a collections in expression.json. All authorized fields must be declared here. All other fields will be silently stripped out on save of the items to those collections. Have a look at expression.json for further details


{
  "title":"This or That",
  "system_name":"this_or_that",
  "version":"1.1.0",
  "collections": [
  {
    "name":"votes", 
    "fields":[
      {
        "name":"thisOne",
        "type":"boolean",
        "operations":["count"]
      },
      {
        "name":"thatOne",
        "type":"boolean",
        "operations":["count"]
      }
    ]
  }
  ]
}


{
  "title":"Trendy",
  "system_name":"trendy",
  "version":"1.1.0",
  "collections": [
  {
    "name":"scores", 
    "fields":[
      {
        "name":"note",
        "type":"number",
        "operations":["average"]
      }
    ]
  }
  ]
}

Once declared in expression.json, you can retrieve a collection, get and set items and display the result of the announced operation.

UT.Expression.ready( function( post ) {
  var votes = post.collection('votes');
  if(votes.getUserItem()){
    // directly display the results.
  } else {
    // display the voting interface
  }

  function onVote(thisOrThat){
    var item = {};
    if(thisOrThat == 'this'){
      item.thisOne = true;
    } else {
      item.thatOne = true;
    }
    votes.setUserItem(item);
    // Now save the data
    post.save();
  }

  function results() {
    var total = votes.count();
    var thisOne = votes.count('thisOne');
    var thatOne = votes.count('thatOne');
    return {
      thisScore: thisOne/total * 100,
      thatScore: thatOne/total * 100
    };
  }
});

UT.Expression.ready( function( post ) {
  var scores = post.collection('scores');
  function results() {
    return scores.average('notes');
  }
});

.setUserItem(value)

Assign the given javascript object as the current user item for this collection. If null or undefined is given, delete the current user item.

Parameters

  • value: an object to assign to the current user

Return

value

Usage

UT.Expression.ready( function( post ) {
  var scores = post.collection('scores');
  scores.setUserItem({note: 5.5, type: 2, comment: "Well done!"});
});

.getUserItem()

Returns the current user item object or undefined if it is not set.

Return

an instance of Object

Usage

UT.Expression.ready( function( post ) {
  var scores = post.collection('scores');
  var entry = scores.getUserItem();
  if(entry){
    // do something
  }
});
