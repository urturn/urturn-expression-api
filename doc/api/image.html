UT.Image

Inherits from the media UT.Media.

Methods

image.crop(options, callback);

Automatically crop an image to specified size.

Take 2 parameters : an hash of options and a callback method called when image is cropped;

Options

{
    x : {int}, // The x position of crop zone on source image
    y : {int}, // The y position of crop zone on source image
    w : {int}, // The width of crop zone on source image
    h : {int}, // The heigh of crop zone on source image
    width : {int}, // The width of final image
    height : {int} // The height of final image
 }

image.autocrop(width, height, callback)

Will crop the image automaticly to the width and height asked. Will try to do the best crop possible given image size and crop size demanded.

image.filter(filters, callback)

Filter the current image.

Take as parameters an array of filters and a callback call when image is filtered. You can use the filter sandbox (inside expression sandbox) to generate this filter array. See Image filters

/!\ Will always filter the original image, if you apply a filter 2 time the result will be the same and not a filtered version of the filtered image.

image.editable(callback)

Parameters

    • callback : a callback function that will be called with the editable image in parameter.

    • Usage

    Will call the callback with a base64 version of image you can use inside a canvas without tainted it.

    UT.Expression.ready(function(post) {
   var image = new UT.Image('http://example.org/image.png');
   image.editable(function(image) {
    var canvas = docuemnt.getElementById('canvas');
    var ctx = canvas.getContext('2d');
    var img = new Image();
    img.src = image;
    ctx.drawImage(img, 0,0);
    // Will not raise DOM security errors
    canvas.toDataURL();
   });
  });
