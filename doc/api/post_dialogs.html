This page describes the various dialogs that can be called from a Post#dialog(type, options, callback)

image

.dialog('image', options, callback);

  • options: 
    {
  size : // crop options
  { 
    width : 23px, // Required width
    height : 23px, // Required height
    adaptUI : true, // True means we should adapt camera screen UI to this size
    auto : true, // Means it will autoCrop to this size
    flexRatio : false // True : when the crop panel is displayed, the user can change the ratio of the crop zone
  },
  applyShaders : 
  [
    ..., // A json array describing filters. See [Filter documentation]()
  ],
  inputType : [  // If specified only first types listed here will be available
    'camera', // The camera
    'search', // The search
    'social', // The link input
    'myUpload' // User uploads
  ]
}

  • callback(String) - will be passed the resulting string or null if no value has been selected.

text

.dialog('text', options, callback)

  • options: 
    {
  value: String      // The default value
  max: Integer       // The maximum number of chars that might be entered
  multiline: boolean // If true, the text input will be multiline
}
  • callback(String) - will be passed the resulting string or null if no value have been selected.

crop

.dialog('crop', options, callback)

Lets you show a crop dialog to crop an image

  • options: 
    {
  image : UT.Image Object,
  size {
    width : 100px,
    height : 100px,
    flexRatio : true
  }
}
  • callback(String) - will be passed the resulting image.
