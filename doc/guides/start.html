From undefined to “Hello World”

Create and deploy your first Expression within 5 minutes.

Install the Urturn toolbelt

To work and deploy an Expression, you will need some specific tools. All of them are available in our command line, working with NodeJS 0.8 or a more recent version.

  1. Prerequisite: install NodeJS
  2. npm install urturn-toolbelt -g

Create an Expression project the hard way

    cd ~/Code
mkdir hello-world
cd hello-world

An Expression basically needs 3 files to run

hello-world/
           /expression.json    --> The Expression descriptor file
           /editor.html        --> Template used to create a new post
           /player.html        --> Template used to display a post

Let's feed those with the following content.

expression.json

    {
    "system_name": "helloworld",
    "title": "Hello World",
    "description": "You know... testing",
    "version": "1.0.0",
    "api_version": "0.6.1"
}

editor.html

    <p><small>editor:</small></p>
<h1 class="title"></h1>
<script>
  UT.Expression.ready(function(post){
    var titleNode = post.node.querySelector('.title');
    // Store content
    post.storage.title = "Hello World!";
    // Ensure that a post is displayed with the same ratio
    post.storage.ratio =
        post.node.offsetHeight / post.node.offsetWidth;

    titleNode.innerHTML = post.storage.title;
    // Save current state
    post.save();
    
    // Posts can be published from that moment on
    post.valid(true);
  });
</script>

player.html

    <p><small>player:</small></p>
<h1 class="title"></h1>
<script>
  UT.Expression.ready(function(post){
    var titleNode = post.node.querySelector('.title');
    // You must call resize at least once in the player
    post.resize(post.storage.ratio * post.node.offsetWidth);
    titleNode.innerHTML = post.storage.title;
  });
</script>

Work with the sandbox

Use the ur command from the toolbelt to start the Sandbox.

    ur sandbox

Visit localhost:3333 to try out your first Expression.

Is it working? Let's deploy it then! Ctrl+C to kill the sandbox.

Deploy your Expression

Use the ur command again.

    ur deploy

You will have to log in to urturn.com using your email and password. Create an account on urturn.com first, if required.

Once the command is finished, your Expression is deployed (but still private). Follow the link in the console output and enjoy your first “Hello World”!
