From undefined to "Hello World" in 5 minutes

Create and deploy your first expression

Install the urturn toolbelt

To work and deploy an expression, you will need some precious tools. All of them are available in our command line, working on top of NodeJS 0.8.x (not compatible with NodeJS 0.10.0 at this time).

Prerequisite: install NodeJS
npm install urturn-toolbelt -g

Create an expression project, the hard way

cd ~/Code
mkdir hello-world
cd hello-world

An expression basically need 3 files to run:

hello-world/
           /expression.json    --> the expression descriptor file
           /editor.html        --> used when someone create a post with your expression
           /player.html        --> used when someone see a post made with your expression

Let's feed those with the following content

expression.json

{
  "system_name": "helloworld",
  "title": "Hello World",
  "description": "You know... testing",
  "version": "1.0.0",
  "api_version": "0.6.1"
}

editor.html

<p><small>editor:</small></p>
<h1 class="title"></h1>
<script>
  UT.Expression.ready(function(post){
    var titleNode = post.node.querySelector('.title');
    post.storage.title = "Hello World!"; // Store content
    post.storage.ratio =                 // to be able to display a post with the same ratio.
        post.node.offsetHeight / post.node.offsetWidth;

    titleNode.innerHTML = post.storage.title;
    
    post.save();                         // save current state
    post.valid(true);                    // the post can be anytime from now.
  });
</script>

player.html

<p><small>player:</small></p>
<h1 class="title"></h1>
<script>
  UT.Expression.ready(function(post){
    var titleNode = post.node.querySelector('.title');
    // You must call resize at least once in the player
    post.resize(post.storage.ratio * post.node.offsetWidth);
    titleNode.innerHTML = post.storage.title;
  });
</script>

Work with the sandbox

Use the ur command from the toolbelt to start a Sandbox 
ur sandbox

Visit http://localhost:3333 to try out your first expression.

It works? so lets deploy it. (<Ctrl+C to kill the sandbox)

Deploy your expression and get famous

Use the ur command again: 
ur deploy

This will ask you to login to urturn.com using your email and password. Create an account on urturn.com first if needed.

Once the command finish, your expression is deployed (but still private). Follow the link in the console output and enjoy your first hello world!
