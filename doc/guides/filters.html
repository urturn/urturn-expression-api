The Urturn API supports the following filters:

  • Brightness
  • Contrast
  • Hue
  • Saturation
  • Sepia
  • Threshold
  • Color Invert
  • Vignette
  • Gamma
  • Exposure
  • Color
  • Noise

Brightness Filter

The Brightness filter lets you change the image brightness.

Parameters:

  • strength: from -1.0 to 1.0, default 0.0 Lets you control how bright the image is. -1.0 is for a black image, 1.0 is for a white image, 0.0 leaves image unchanged.

example: json { "filter" : "brightness", "parameters" : { "strength" : 0.2 } }

Contrast

The Contrast filter lets you change the image contrast.

Parameters:

  • strength: from -1.0 to 1.0, default 0.0 Lets you control the level of contrast. -1.0 is for a grey image, 1.0 is for a high contrast, 0.0 leaves image unchanged.

example: json { "filter" : "contrast", "parameters" : { "strength" : 0.2 } }

Hue

The Hue filter let you change the hue of the image.

Parameters:

  • strength: from -1.0 to 1.0, default 0.0 Lets you control the hue. 0.0 leaves image unchanged.

example: json { "filter" : "hue", "parameters" : { "strength" : 0.2 } }

Saturation

The Saturation filter lets you change the image saturation.

Parameters:

  • strength: from -1.0 to 1.0, default 0.0 Lets you control the saturation. 0.0 leaves image unchanged. -1.0 gives you a desaturated (black and white) image, 1.0 an over-saturated one.

example: json { "filter" : "saturation", "parameters" : { "strength" : 0.2 } }

Sepia

The Sepia filter lets you apply a yellow-black filter on your image.

Parameters:

  • strength: from 0.0 to 1.0, default 0.0

    Lets you control sepia levels. 0.0 leaves image unchanged. 1.0 applies a full sepia filter

example: json { "filter" : "sepia", "parameters" : { "strength" : 0.2 } }

Threshold

The Threshold filter transforms your image to a 2-color image.

Parameters:

  • strength: from 0.0 to 1.0, default 0.5

    Lets you control the threshold limit. 0.5 means all colors under 128 will be primary, all colors up of 128 will be secondary.

  • alpha: from 0.0 to 1.0, default 1.0

    Lets you control the alpha of this filter.

  • primaryColor: array of 3 numbers defining the RGB channels [r (0-255),g (0-255),b (0-255)], default [0,0,0];

    Lets you define the primary color.

  • secondaryColor: array of numbers defining RGB channels[r (0-255),g (0-255),b (0-255)], default [255,255,255];

    Lets you define the secondary color.

example: json { "filter" : "threshold", "parameters" : { "strength" : 0.2, "alpha" : 0.5, "primaryColor" : [24,44,33], "secondaryColor" : [223,222,123] } }

Color invert

Invert color on your image.

Parameters:

  • No Parameters!

example: json { "filter" : "invertColor" }

Vignette

Add a Vignette on top of your image

Parameters:

  • amount: from 0.0 to 1.0, default 0.0 Amount of vignettes added to your image, 0.0 no vignette, 1.0 maximum amount

  • size: from 0.0 to 1.0 Size of the vignette border: 0.0 no vignette, 1.0 maximum size

  • centerX: from 0.0 to 1.0, default 0.5 X coordinate of the center of your vignette. 0.0 means completely to the left, 1.0 completely to the right of your image. 0.5 means exactly in the middle of your image.

  • centerY: from 0.0 to 1.0, default 0.5 Y coordinate of the center of your vignette. 0.0 means right at the top, 1.0 right at the bottom of your image. 0.5 means exactly in the middle of your image.

  • r: from 0 to 255, default 0 The red channel of vignette color

*g: from 0 to 255, default 0 The green channel of vignette color

*b: from 0 to 255, default 0 The blue channel of vignette color

example: json { "filter" : "vignette", "parameters" : [ "size" : 0.2, "amount" : 0.3, "centerX" : 0.5, "centerY" : 0.5, "r" : 12, "g" : 234, "b" : 123 ] }

Gamma

The Gamma filter lets you change the gamma of your image.

Parameters:

  • strength: from 0.0 to 3.0, default 1.0

    Lets you control the gamma strength. 0.0 image is white. 1.0 image intact. 3.0 Strong Gamma

example: json { "filter" : "gamma", "parameters" : { "strength" : 0.2 } }

Exposure

The Exposure filter lets you change the exposure of your image.

Parameters:

  • strength: from 0.0 to 1.0, default 0.0

    Lets you control the exposure. 0.0 image is untouched. 1.0 image is over-exposed.

example: json { "filter" : "exposure", "parameters" : { "strength" : 0.2 } }

Color

The color filter lets you change the color balance of your image.

Parameters:

  • a: from 0.0 to 1.0, default 0.0

    Lets you control alpha of this filter

  • r: from 0.0 to 1.0, default : 0.0

    Lets you control the influence of this filter to the red channel.

  • g: from 0.0 to 1.0, default : 0.0

    Lets you control the influence of this filter to the green channel.

  • g: from 0.0 to 1.0, default : 0.0

    Lets you control the influence of this filter to the blue channel.

example: json { "filter" : "color", "parameters" : { "a" : 0.2, "r" : 0.3, "g" : 0.7, "b" : 0.4 } }

Noise

Add noise to your image.

Parameters:

  • Amount: from 0.0 to 1.0, default 0.0

    Lets you control the amount of noise in your image. 0.0 image is untouched. 1.0 lots of noise.

example: json { "filter" : "noise", "parameters" : { "amount" : 0.2 } }
